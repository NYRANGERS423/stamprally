# Accolade seed suggestions

Reference list of starter accolade ideas — not used by the code. Useful as
a copy-paste source when seeding `/admin/accolades` for a new deployment.

These were originally the hardcoded `SUGGESTED_ACCOLADES` constant in
`src/lib/accolades.ts`. The catalog now lives in the `AccoladeTemplate`
DB table (admin-managed at `/admin/accolades`), so this is purely a
reference document.

| Emoji | Label | Description |
|---|---|---|
| 🏆 | Champion of the Day | Top stamper at an event. |
| 🌟 | MVP | Most valuable participant. |
| 🎯 | Team Spirit | Rallied colleagues throughout the event. |
| 🚀 | First Adopter | Among the first to sign up. |
| 💪 | Volunteer Hero | Went above and beyond to help out. |
| 🎟️ | Special Guest | VIP visitor recognition. |
| 🌱 | Earth Day Champion | Top performer at Earth Day. |
| 🚢 | Captain's Pick | Manager's chosen recognition. |
| 🎨 | Creative Spark | Most creative engagement. |
| 🔥 | On Fire | Streak of activity participation. |
