## Format guidance

- **PNG preferred** for crisp UI; JPEG fine for photo-heavy passport cards
- **Mobile viewport** for everything except #8 (admin event detail), which is desktop
- Recommended mobile size: 390×844 (iPhone stock) — captures the same density the audience sees
- Don't bother with device-frame mockups — flat captures are easier to compare against design proposals
- File size sanity: keep each under ~500 KB; compress with [tinypng.com](https://tinypng.com) or `pngquant` if needed
